HiDT

High-Resolution Daytime Translation. Try it!

Git Hub repository : HiDT repo
Testing API : on Ainize

Usage

1. Please name the image file in English or in numbers.

2. Please select the daytime you want.

3. Please upload the landscape image file you want to change the daytime.

4. Please press the Run button and wait for about 5 seconds.

Upload

* Please upload an image file that is less than 2MB.
